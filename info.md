# Light Manager

Displays all currently "on" lights in a custom sidebar panel.

## Features

- Simple table view of active lights
- Real-time updates when lights turn on/off
- Clean integration with Home Assistant UI
- Dark and light theme support
- No configuration required

## Installation

1. Install via HACS
2. Restart Home Assistant
3. Navigate to "Light Manager" in the sidebar

## Usage

Open the "Light Manager" panel from the sidebar to see all lights that are currently on. The table will automatically update when lights change state.
